# 写给普通人看的网页开发课

零基础入门Web开发，手把手教你搭建一个简单的博客系统。

通过这门课，你将学会：

- HTML5基础知识：
  - 语义化标签
  - 属性
  - 页面结构

- CSS基础知识
  - 选择器
  - 元素盒子
  - 布局和样式

- JS基础知识
  - 浏览器中的JavaScript

- 实现一个简单的个人博客并通过Github Pages发布
